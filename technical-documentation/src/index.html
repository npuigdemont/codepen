Introducció

React va ser dissenyat des del principi perquè es pogués adoptar de forma gradual, i poder fer servir només algunes coses de React. Pots afegir una miqueta d’interactivitat a una pàgina ja existent. Els components de React son una manera de fer això.

La majoria de llocs web no son, i no necessiten ser, aplicacions d’una sola pàgina. Amb unes poques línies de codi i sense eines de recopilació, pots provar React en llocs petits del teu c web. I des d'aquí pots expandir la teva presència de forma gradual, o mantenir algun widget dinàmic.

Conceptes Javascript necessaris

Necessito realment utilitzar un framework com React per desenvolupar aquesta aplicació?

La resposta a aquesta pregunta dependrà de l'aplicació, de quant temps tens per desenvolupar-la, de com i per qui haurà de ser mantinguda després i probablement de molts altres factors conjunturals. Però el més important és que no sabràs respondre-correctament si no coneixes bé el llenguatge, en aquest cas javascript.

Així que la meva proposta és, aprèn javascript fins que tu mateix siguis capaç de respondre a la meva pregunta, perquè sàpigues el que implica desenvolupar tota l'aplicació sense frameworks, i al llegir sobre el que fa React (per exemple) et sembli que és convenient utilitzar-lo, tenint en compte totes les implicacions que té per al projecte.
És veritat cada framework té les seves peculiaritats pròpies, però hi ha alguns desavantatges comuns a tots.

  • La majoria lliguen l'arquitectura de el projecte al framework. Si has de mantenir-lo, actualitzar-lo, modificar-lo o reaprofitar-ne parts en altres projectes, hauràs de fer-ho fent servir el framework, perquè fer-ho sense significarà gairebé amb total certesa reescriure bona part de el codi...
  • Si més gent col·labora en el projecte, tots hauran de dedicar temps a aprendre el framework (si no el coneixien abans)

No és que em sembli malament aprendre'n ni fer-los servir, però les llibreries i frameworks van i vénen, es posen de moda o cauen en l'oblit, el llenguatge dura molt més.
Alguns llenguatges també cauen en desús amb el temps, però aprendre un llenguatge en profunditat implica aprendre a programar cada vegada millor en el procés, mentre que aprendre frameworks abans d'aprendre bé el llenguatge t'ensenya a prendre dreceres, i el que és pitjor moltes vegades sense saber de què estàs fent una drecera.
Per cert, quan coneixes bé un llenguatge, aprendre a utilitzar un framework basat en ell no porta massa temps.

Afegir react al web

Ara aprendrem com afegir un component de React a una pàgina HTML que ja existeix. Es poden seguir els passos a la teva web o bé crear un nou arxiu HTML per poder practicar.

No hi ha cap necessitat de fer servir cap requeriment per fer la instal·lació només:

  • Connexió a internet
  • una estona per aprendre com funciona

Afegir contenidor DOM

Per començar, obre la pàgina HTML que vulguis editar. Afegeix una etiqueta < div> buida per marcar el lloc on vols veure alguna cosa amb React. Per exemple: 

 < !-- ... HTML existent ... -->
< div id="like_button_container">
            < !-- ... HTML existent ... -->
a aquest < div> li afegim un atribut HTML id que és unic. Això ens premetrà trobar-lo des del codi Javascript més endavant i visualitzar un component de React dins d'aquest.

Consell: Pots afegir un "contenidor" < div> com aquest en qualsevol lloc dins l'etiqueta < body>. Pots tenir la quantitat de contenidors independents en el DOM que vulguis. Normalment estan buits ja que React reemplaça el contingut existent dins els contenidors del DOM.

Afegir etiquetes script

El pas següent és afegir tres etiquetes < script > a la pàgina HTML just abans de tancar l'etiqueta < /body>:
< !-- ... més HTML ... -->
< !-- Carregar React. -->
< !-- Nota: quan es desplegui, reemplaçar "development.js" amb "production.min.js". -->
< script src="https://unpkg.com/react@16/umd/react.development.js" crossorigin>
< script src="https://unpkg.com/react-dom@16/umd/react-dom.development.js" crossorigin> < !-- Carreguem el nostre component de React. --> < script src="like_button.js">
< /body>
Les primeres dues etiquetes carreguen React. La tercera carrega el teu codi del component.

Crea component React

Crea un arxiu que es digui like_button.js al mateix lloc on tens l'arxiu HTML.

  • Obre aquest codi inicial i enganxa'l a l'arxiu que acabes de crear.
  • Després al codi inicial afegeix les següents dues línies al final de like_button.js: const domContainer = document.querySelector('#like_button_container'); ReactDOM.render(e(LikeButton), domContainer);
Aquestes dues línies de codi troben el < div> que afegim al nostre HTML al primer pas i mostren el component de React per el nostre botó "Like" a dins.
